Spindle Unstable Update Site

This site contains feature and plug-in versions for Spindle 3.0 & 3.1 unstable releases, published in the form expected by the Eclipse Update Manager.

THIS IS NOT YET PRODUCTION QUALITY CODE. IT IS MADE AVAILABLE HERE IN THE HOPES THAT YOU WILL TRY IT OUT AND POST BUGS.

IMPORTANT:

3.0 releases of Spindle are not warranted to work with any build of Eclipse 3.0.

3.1 releases of Spindle will only install in Eclipse 3.0.

bugs can be logged here

Use the Update Manager's Feature Updates view to open this site (http://spindle.sourceforge.net/unstable) to locate new and updated Spindle project feature versions to download.

This is the only source unstable builds of Spindle. The thinking is that releasing to sourceforge is slow and cumbersome. We hope to be able to release more often here while the code is still shiny new.

Select builds from this site may be classified as beta builds and will also be made available via Eclipse Update at http://spindle.sourceforge.net/updates

JettyLauncher has moved to a new sourceforge project.
