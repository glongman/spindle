Running a Project

Once your project builds corectly in Eclipse, it is ready to be run. You have a couple of options for running the project.

  • Running using Jetty Launcher This is the simplest way, and it allows you to debug your application in Eclipse!
  • Manually Deploying You can deply your app to any servlet container. Just make a .war file starting from within the content folder.

    cd project/context jar cvf sample.war *.*

    And then deploy to Tomcat, or any other servlet container, as you would any .war file. You may need to add the Tapestry Libraries to your servlet container or your .war file, depending on how you want to set things up.
  • Running Using Tomcat Launcher http://www.sysdeo.com/eclipse/tomcatPlugin.html I haven't tried the tomcat launcher, but it could work.

We recommend using Jetty Launcher, as it is very easy to use.
