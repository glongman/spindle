Running Your Project with Jetty

Jetty is an open source web server and servlet container that you can use to run your Tapestry applications.

Jetty Launcher is an Eclipse plug-in that lets you easily run and debug Tapestry applications from with Eclipse, using Jetty.

First install Jetty and Jetty Launcher. It will only take a couple minutes.

Running the Project

First make sure you are in the Java Perspective. If you don't, the Run menu won't have the correct entries in it.

  • On the Window menu
  • Select Open Perspective
  • Select Java

To run your project:

  • Open the Run menu
  • Select Run…
  • Select Jetty Web in the Configurations section
  • Click New
  • Enter “Tapestry Sample App”, if prompted (varies by Eclipse version)
  • Under Jetty Configuration, click Browse, and find your jetty installation directory.
  • Make sure the port number is OK If you’re running a web server already, you may need to change the port number Jetty uses. This is under Use Options, http port on this tab.
  • Click the Run button
  • In your browser, go to http://localhost:8080/app Or use whatever port you specified, instead of 8080

You should see “Tom’s Sample Tapestry Application”, or whatever string you entered above.

If there is a problem, check in Eclipse's console window. The most common problem is a port conflict.

If you have Jetty Launcher installed, then you might also want to review the Jetty Launcher integrated help (also available online).
