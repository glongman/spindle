Creating a Project

Creating a Tapestry project with Spindle is just like creating any other project.

  • Open the File menu
  • Select New
  • Select Project...
  • Select and expand Tapestry
  • Select Tapestry Web Project
  • Click Next >
  • In the New Tapestry Web Project dialog box, enter a project name “Tapestry Sample Project”
  • Click Next >
  • Click Finish, and your project will be created

Building the Project

You may see that the name of your project has a comment after it in the Eclipse Navigator window. This may be “[Tapestry Sample Project.application]” or “[Not built yet]”.

<>If you see [Not built yet], your app has not successfully been built. This may take a few seconds. Since this is a fresh outline, it should build, if you have set Eclipse to build automatically, or you may have to do a build manually.. It may also not build if you have errors in your application.

Adding Something

The default Tapestry application that Spindle generates has nothing in it. Let's add one string, to the home page, so we can see it when we run the application.

In the Eclipse Navigator tab

  • Select and expand Tapestry Sample Project
  • Expand the context folder
  • Expand the WEB-INF folder
  • In the WEB-INF folder, double click on the file Home.html
  • In the editor that opens just add a line of text between <body> and </body>, like "Tom’s Sample Tapestry Application” and save the file

You now should have a working Tapestry application. The easiest way to run it is to use the Jetty Launcher. See Running a Project.
