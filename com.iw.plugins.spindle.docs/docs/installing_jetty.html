Installing Jetty

Jetty is an open source Servlet Container, which can be used with Jetty Launcher to run & debug your Tapestry Applications (and other Servlets) from within Eclipse.

Jetty is not required to use Spindle, it just makes your life easier.

Installing Jetty

Jetty is available from:

http://jetty.mortbay.org/jetty/

Just download and unzip, and you are ready to go. No configuration is needed. You may want to do some later, for more advanced uses of Jetty, but to start with no configuration is needed.
