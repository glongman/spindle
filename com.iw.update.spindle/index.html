Use the Update Manager's Feature Updates view to open this site (http://data.intelligentworks.com) to locate new and updated Spindle project feature versions to download.

This is the only source for Spindle 3.0 integration releases. The thinking is that releasing to sourceforge is slow and cumbersome. We hope to be able to release more often here while the code is still shiny new.

Select builds from this site may be classified as beta builds and will also be made available via Eclipse Update at http://spindle.sourceforge.net/updates
