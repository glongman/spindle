Installing Spindle

If you are viewing this from within Eclipse’s help system then congratulations! You have successfully installed Spindle. Please continue with the next page.

If you are browsing this documentation on the web, then we’ll take you through the steps on installing Spindle. It’ll only take a minute (plus download time).

Spindle is installed using the Eclipse plug-in update mechanism from:

http://spindle.sourceforge.net/unstable/


Step-by-Step Intallation Instructions:

  • Start Eclipse
  • Select Help menu
  • Select Software Updates
  • Select Find and Install
  • Check Search for New Features to Install
  • Click Next >
  • Click New Remote Site
  • Name it Spindle Unstable
  • URL is http://spindle.sourceforge.net/unstable/
  • Click OK
  • Check Spindle Unstable
  • Click Next >
  • Put a check by the latest version of Spindle
  • Click Next >
  • Accept the license agreement
  • Click Next >
  • Click Finish
  • Click Install, (ignoring the warning about “unsigned feature”)

Stable vs. Unstable Versions

Note that there are two update sites for Spindle, the 'stable', and 'unstable' versions. You can get the latest stable version from http://spindle.sourceforge.net/updates. I prefer the unstable, since they have been quite stable, and the stable version is quite old.
