Installing Spindle

If you are viewing this from within Eclipse’s help system then congratulations! You have successfully installed Spindle. Please continue with the next page.

If you are browsing this documentation on the web, then we’ll take you through the steps on installing Spindle. It’ll only take a minute (plus download time).

There are two ways to install Spindle, online and using an archived update site that you download from SourceForge.

Either way, Spindle is installed using the Eclipse plug-in update mechanism. For more information about the update mechanism check out its documentation.

If you are viewing this document in Eclipse, the detailed help for update is available in the tree of topics on the left.

Workbench User Guide > Tasks > Updating Features with the update manager

The same topic is available online.

Online Installing

The update URL for spindle is:

http://spindle.sourceforge.net/updates


Step-by-Step Intallation Instructions:

  • Start Eclipse
  • Select Help menu
  • Select Software Updates
  • Select Find and Install
  • Check Search for New Features to Install
  • Click Next >
  • Click New Remote Site
  • Name it Spindle
  • URL is http://spindle.sourceforge.net/updates/
  • Click OK

Things will occur in the background. In short order a new dialog titled "Updates" will appear

  • Check Spindle
  • Click Next >
  • Put a check by the latest version of Spindle
  • Click Next >
  • Accept the license agreement
  • Click Next >
  • Click Finish
  • Click Install.

    • (Spindle is unsigned so it is your choice to ignore the warning about “unsigned feature”. Be aware that if you do not accept, Spindle will not be installed.)

Installing from a downloaded archive

Often it is convenient to download an archived version of the Spindle update site and install from that. Archives are available from the SourceForge file release system.

zip files downloaded from SourceForge are acutally Eclipse Update sites themselves and not zipped plugins. You can't unzip these files into an Eclipse installation!

Step-by-Step Intallation Instructions:

  • Download the archived site of you choice from SourceForge.
  • Start Eclipse
  • Select Help menu
  • Select Software Updates
  • Select Find and Install
  • Check Search for New Features to Install
  • Click Next >
  • Click New Archived Site
  • Choose the archive you downloaded using the choose dialog that opened.
  • Name it Spindle
  • Click OK

Things will occur in the background. In short order a new dialog titled "Updates" will appear

  • Check Spindle
  • Click Next >
  • Put a check by the version of Spindle that appears in the tree.
  • Click Next >
  • Accept the license agreement
  • Click Next >
  • Click Finish
  • Click Install.

    • (Again, Spindle is unsigned so it is your choice to ignore the warning about “unsigned feature”. Be aware that if you do not accept, Spindle will not be installed.)
