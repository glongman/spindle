Installing Jetty Launcher

Jetty Launcher is an Eclipse plug-in that lets you easily run and debug Tapestry applications from with Eclipse. Jetty Launcher is not required to use Spindle, it just makes it better, since you can then debug your new app within Eclipse. Of course, you need to install Jetty to use Jetty Launcher.

Installing Jetty Launcher

Jetty Launcher is installed via the Eclipse plug-in update mechanism. You can install Jetty Launcher from:

http://jettylauncher.sourceforge.net/updates

Step by Step Directions:

  • Start Eclipse
  • Select Help menu
  • Select Software Updates
  • Select Find and Install
  • Check Search for New Features to Install
  • Click Next >
  • Click New Remote Site
  • Name it Jetty Launcher
  • URL is http://jettylauncher.sourceforge.net/updates
  • Click OK
  • Check Jetty Launcher
  • Click Next >
  • Check Jetty Launcher Plugin
  • Click Next >
  • Accept the license agreement
  • Click Next >
  • Click Finish

When you run your first Tapestry app with Eclipse, you will have to do a bit more configuration for Jetty Launcher in the "Run" dialog box. This will just be things like browsing to the jetty install folder, and choosing an http port.
