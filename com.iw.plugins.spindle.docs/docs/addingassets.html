Adding Images and Other Assets

Now let's add an image to the project.

To add an image we need to:

  • Copy the image to the place in your project where you want to the image to reside.
    For example, the "context/images" directory.
  • Declare the asset in the .page specification
  • Add an <IMG> tag to the html file that will display the image.

Add the Image to the .page File

To add an image to the page specification, open Home.page

In the Eclipse Package Explorer tab:

  • Select and expand Tapestry Sample Project
  • Expand the context folder
  • Expand the WEB-INF folder
  • In the WEB-INF folder, double click on the file Home.page 
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE page-specification PUBLIC
  "-//Apache Software Foundation//Tapestry Specification 3.0//EN"
  "http://jakarta.apache.org/tapestry/dtd/Tapestry_3_0.dtd">
<!-- generated by Spindle, http://spindle.sourceforge.net -->

<page-specification class="org.apache.tapestry.html.BasePage">
    <description>add a description</description>

</page-specification>

Then, in the editor for Home.page

  • On the blank line after </description>
  • Hit control-space to bring up the Spindle context menu.
  • Scroll down to "context-asset (empty)" and hit enter.

You should see something like this:

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE page-specification PUBLIC
  "-//Apache Software Foundation//Tapestry Specification 3.0//EN"
  "http://jakarta.apache.org/tapestry/dtd/Tapestry_3_0.dtd">
<!-- generated by Spindle, http://spindle.sourceforge.net -->

<page-specification class="org.apache.tapestry.html.BasePage">
    <description>add a description</description>
  <context-asset name="name" path="value"/>  
</page-specification>

Enter the name and value attributes. The name can be anything, we'll call it "photo". The value is the path from the context folder to your asset. If the image is "context/images/photo.jpg", value will be "images/photo.jpg"

<context-asset name="photo" path="image/photo.jpg"/>

Add the Image to the .html File

  • Now open Home.html
  • Add an <img> tag anywhere in the body.
    <img jwcid="@Image" image="ognl:assets.photo"/>
  • Run the program to see your image

Pop Ups

Go back to Home.page for a moment, and hover your mouse over context-asset. Notice that a pop up window opens, describing the tag's purpose, and listing the attributes of the tag.
