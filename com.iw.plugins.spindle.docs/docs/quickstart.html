Spindle Quick Start

This page is the really quick guide to Spindle. You can accept all defaults and be running in 12 minutes, if you have a fast connection. Yes, I timed it. :-)

For a more detailed walk-through, please continue to the next page.

  • Install Spindle using Eclipse Plug-in manager (http://spindle.sourceforge.net/unstable)
  • Install Tapestry (http://jakarta.apache.org/tapestry/)
  • Download & unzip Jetty (http://jetty.mortbay.org/jetty/)
  • Install Jetty Launcher using Eclipse Plug-in manager (http://jettylauncher.sourceforge.net/updates)
  • File menu/ New/ Project.../ Tapestry Project to create your project
  • Add "Hello, world" to Home.html, so your app isn't a blank screen.
  • Run your application from within Eclipse[details]
    • Run menu/ Run... (Hint: Switch to Java Perspective!)
    • Run Dialog: Configurations/Jetty Web/New/Name it something
    • Run Dialog: Jetty Home: Browse Button, find the home.
    • Run Dialog: Click the Run button
  • Open a browser window to http://localhost:8080/app to see your "Hello, world"
