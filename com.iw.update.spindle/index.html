Spindle Update Site

This site contains feature and plug-in versions for Spindle published in the form expected by the Eclipse Update Manager.

IMPORTANT:

3.2 releases of Spindle will only install in Eclipse 3.1.X

3.1 releases of Spindle will only install in Eclipse 3.0.X

3.0 releases of Spindle will only install in Eclipse 2.1.X

bugs can be logged here

Use Eclipse feature update facility and the url of this page(http://spindle.sourceforge.net/updates) to locate new and updated Spindle project feature versions to download.
